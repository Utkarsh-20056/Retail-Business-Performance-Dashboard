# 📊 Retail Sales Performance Dashboard | Microsoft Excel

An interactive Excel dashboard built to analyze retail sales performance and transform raw sales data into meaningful business insights using Pivot Tables, Pivot Charts, Slicers, and KPI Cards.

---

## 📌 Project Overview

This project focuses on analyzing annual retail sales data to identify sales trends, customer behavior, regional performance, and product performance. The dashboard enables users to interactively explore the data through slicers and dynamic visualizations.

---

## 🚀 Dashboard Features

- 💰 KPI Cards
  - Total Sales
  - Total Orders
  - Average Order Value
  - Delivered Orders
  - Returned Orders
  - Cancelled Orders
  - Delivery Rate
  - Return Rate
  - Cancellation Rate

- 📈 Interactive Visualizations
  - Monthly Sales Trend
  - Revenue by Gender
  - Revenue by State
  - Revenue by Platform
  - Revenue by Product Category
  - Revenue by Order Value

- 🎛 Interactive Slicers
  - Month
  - Category
  - Channel

- 📌 Business Insights Panel

---

## 📊 Key Business Insights

- Women contributed nearly **64%** of total revenue.
- Maharashtra generated the highest sales among all states.
- Amazon was the top-performing sales platform.
- The **Set** category generated the highest revenue.
- Sales reached their peak during **March**.
- Delivery success rate remained above **92%**.
